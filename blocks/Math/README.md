# Math

Math blocks perform **arithmetic, vector operations, CFrame manipulation, Color3 construction, and random number generation**.

---

## Overview

| Block Type | Description |
|---|---|
| `Addition` | Add two numbers |
| `Subtraction` | Subtract two numbers |
| `Multiplication` | Multiply two numbers |
| `Division` | Divide two numbers |
| `Modulus` | Remainder of division |
| `Exponentiation` | Raise to a power |
| `SolveEquation` | Evaluate a math expression string |
| `RandomNumber` | Generate a random number in a range |
| `Absolute` | Absolute value |
| `Floor` / `Ceil` / `Round` | Rounding operations |
| `Clamp` | Clamp a value between min and max |
| `Vector3New` | Construct a Vector3 |
| `CFrameNew` | Construct a CFrame |
| `Color3New` | Construct a Color3 |

## Number Format

All number literals in VS scripts are **hex-encoded**:

| Hex | Decimal |
|---|---|
| `A` | 10 |
| `F` | 15 |
| `32` | 50 |
| `64` | 100 |
| `C8` | 200 |
| `1F4` | 500 |
