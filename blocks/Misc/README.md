# Miscellaneous

Miscellaneous blocks cover **utilities, debug tools, services, JSON serialization, and plugins** — anything that doesn't fit neatly into the other categories.

---

## Overview

| Block Type | Description |
|---|---|
| `Print` | Print a message to the VS output console |
| `Warn` | Print a warning message |
| `Error` | Throw an error |
| `Comment` | Add a non-executing comment node to the canvas |
| `JSONEncode` | Serialize a table to a JSON string |
| `JSONDecode` | Deserialize a JSON string to a table |
| `HttpGet` | Make an HTTP GET request |
| `HttpPost` | Make an HTTP POST request |
| `GetDataStore` | Access a DataStore by name |
| `SetDataStoreValue` | Save a value to a DataStore |
| `GetDataStoreValue` | Load a value from a DataStore |
