# Examples

This section provides practical VisualSource script examples for common game patterns. Each example shows the decoded human-readable format, which you can copy into the VS editor or encode using the Encoder tab.

---

## How to Use These Examples

1. Copy the decoded script text
2. Open RetroStudio and navigate to your script
3. Paste it into the Decoder to visualize it, or use it directly
4. The Encoder will compress it automatically when saving

---

## Example Categories

- [Player Events](player-events.md) — Join/leave handling, leaderstats, character loading
- [Variables & Math](variables-math.md) — Arithmetic, random numbers, string operations
- [Loops](loops.md) — For loops, while loops, looping through children
- [Properties](properties.md) — Getting and setting object properties

---

## Script Format Reminder

All examples start with an `Editor` header and use two spaces between blocks:

```
Editor CameraPosition 0,0 CameraZoom 0.1C2  Block Type ... Name ...  Block Type ... Name ...
```

Refer to the [Script Format](../core-concepts/script-format.md) page for full details.
